## Featured Project

### ECommerce Microservices System
**Repository:** https://github.com/mahaidang/ecommerce.git  

**Tech Stack:**  
.NET 8, YARP API Gateway, CQRS, MediatR, RabbitMQ (Saga),
gRPC, SQL Server, MongoDB, Redis, JWT, RBAC, Sepay, Docker

**Overview:**  
A scalable **microservices-based eCommerce backend system** designed to
handle real-world order processing, payment confirmation, and inventory
consistency.

**Key Highlights:**
- **YARP API Gateway** as a single entry point with JWT & RBAC security
- **Distributed Saga pattern** using RabbitMQ for reliable order processing
- **gRPC communication** for synchronous inventory validation
- **Polyglot persistence** with SQL Server, MongoDB, and Redis
- **Sepay webhook integration** with idempotent payment handling
- **Docker Compose** for local development and deployment consistency
